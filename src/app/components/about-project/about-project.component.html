Project: Dangers of Phishing

This project focuses on raising awareness about the dangers of phishing attacks.

Frontend Repository:

Frontend GitHub Repository

Backend Repository:

Backend GitHub Repository

Project Details:

  • Built with Angular for the frontend.
  • Utilizes AWS SAM (Serverless Application Model) for the backend.
  • Backend components include AWS Lambda, API Gateway, and DynamoDB.

Objective:

The main goal of this project is to educate individuals about the risks associated with phishing attacks. It includes:

  • Real-world examples of phishing pages, including mimicked pages of popular services like Google and Facebook to showcase the potential dangers.
  • Tips and best practices for avoiding falling victim to phishing scams.
